# oneWordStoryBot

## A Discord Bot allowing Users to write a story, one word at a time.

### Install dependencies:
```
npm i
```

### Create `.env`:
```env
TOKEN=YOUR_BOT_TOKEN_HERE
CHANNEL=1207024002296455168
USERBLOCK=3
VERIFY=yes
```

### Run the bot script:
```
node main.js
```
